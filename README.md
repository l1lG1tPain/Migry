# 🧠 Мигри — трекер головной боли v1.0

Персональный PWA-дневник мигрени. Помогает фиксировать приступы, отслеживать триггеры и симптомы, а также формировать отчёт для врача.

---

## ✨ Возможности

- **Дневник приступов** — интенсивность, тип боли, локализация, симптомы, триггеры, лекарства
- **Календарь** — цветовая индикация дней по силе боли (слабая / средняя / сильная)
- **Инсайты** — появляются после 7 дней записей: частота, триггеры, паттерны, предупреждения
- **Импорт из MigreBot** — загрузка истории из `.xlsx` экспорта приложения MigreBot
- **Отчёт для врача** — текстовая выгрузка с ключевыми данными, копируется в буфер
- **Тёмная и светлая тема** — переключение в один клик
- **PWA** — устанавливается на домашний экран, работает офлайн
- **Хранилище** — все данные локально в IndexedDB, ничего не уходит на сервер

## 📊 Формула аналитики

Движок аналитики основан на рекомендациях **ICHD-3** и опубликованных исследованиях.

| Метрика | Формула |
|---|---|
| Частота | `(кол-во приступов / дни) × 30` |
| Средняя интенсивность | `Σ интенсивность / кол-во приступов` |
| Риск триггера | `P(T\|H=1) = приступы с триггером / все приступы × 100` |
| Корреляция | Pearson r (r > 0.3 = значимый триггер) |

**Пороги частоты:**
- `< 8 / мес` — эпизодическая форма ✅
- `≥ 8 / мес` — частая, рекомендована профилактика ⚠️
- `≥ 15 / мес` — хроническая мигрень 🚨

**Генерируемые инсайты (11 типов):**
- Частота и форма мигрени
- Интенсивность боли
- Топ триггеры с процентами
- Паттерн времени суток
- Эффективность лекарств
- Риск медикаментозной головной боли (> 10 доз / 10 дней)
- Зрительная мигрень (аура ≥ 30%)
- Менструальная мигрень
- День недели / мигрень выходного дня
- Нестабильный сон как триггер
- Сенсорная чувствительность (шум + запахи)

## 📥 Импорт из MigreBot

Приложение принимает `.xlsx` файл из экспорта **MigreBot** (Telegram-бот).

**Поддерживаемые колонки:**

| # | Колонка | Маппинг |
|---|---|---|
| 0 | Дата | `date` |
| 1 | Время | — |
| 2 | Головная боль | Только строки «Да» импортируются |
| 3 | Признаки мигрени | `symptoms.aura` |
| 4 | Менструальный цикл | `symptoms.menstruation` |
| 5 | Принятые медикаменты | `medications[].name / dose / effectiveness` |
| 6 | Интенсивность боли | `intensity` (1–10) |
| 7 | Локализация | `location` + `side` |
| 8 | Характер | `painType` |
| 9 | Нагрузки | `triggers.physActivity` |
| 10 | Тошнота | `symptoms.nausea` |
| 11 | Фотофобия | `symptoms.photophobia` |
| 12 | Фонофобия | `symptoms.phonophobia` |
| 13 | Триггеры | `triggers.*` (текстовый разбор) |
| 14 | Начало боли | `startTime` |
| 15 | Окончание боли | `endTime` |
| 16 | Комментарии | `notes` |

## 🗄 Схема данных (IndexedDB)

### Store: `attacks`

```js
{
  id:          Number,       // autoIncrement
  date:        'YYYY-MM-DD',
  startTime:   'YYYY-MM-DDTHH:MM',
  endTime:     'YYYY-MM-DDTHH:MM' | null,
  timeChip:    'morning' | 'afternoon' | 'evening' | 'night' | 'ago1' | 'ago3' | 'ago5' | null,
  intensity:   1–10,
  painType:    'throbbing' | 'pressing' | 'sharp' | 'aching' | 'bursting' | 'squeezing' | 'dull' | null,
  location:    'temples' | 'forehead' | 'back' | 'full' | 'top' | 'eyelids' | 'jaw' | null,
  side:        'left' | 'right' | 'both' | null,
  symptoms: {
    nausea, vomiting, photophobia, phonophobia,
    aura, dizziness, numbness, weakness, menstruation
  },
  triggers: {
    stress, lessSleep, moreSleep, hunger, weather,
    smell, noise, runnyNose, screen, physActivity,
    alcohol, caffeine, hookah
  },
  medications: [{ name, dose, qty, effectiveness }],
  notes:       String,
  createdAt:   timestamp
}
```

## ⚕️ Дисклеймер

Мигри — инструмент самонаблюдения. Не является медицинским устройством и не ставит диагнозов. Все данные хранятся локально на устройстве. Отчёт предназначен для передачи лечащему врачу.
